25 /*
26  * The `then` function implemented below is designed to be very similar to JavaScript's Promise.then in functionality.
27  *
28  * You provide it with a promise or future/shared_future object, and it will invoke a callback with the result of that promise or future.
29  *
30  * Additionally, it will resolve recursive futures. As in, if the original future resolves to another future, and so forth.
31  *
32  * It will also resolve futures/promises returned by the callback, so the future returned by `then` will always resolve to a non-future type.
33  *
34  * Basically, you can layer up whatever you want and it'll resolve them all.
35  * */
