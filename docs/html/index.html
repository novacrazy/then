This is a generic implementation of then for C++14, allowing you to chain together futures, promises and so forth easily and efficiently.

## Example:

1 {C++}
2 #include "thenable.hpp"
3 
4 #include <iostream>
5 #include <chrono>
6 #include <thread>
7 #include <assert.h>
8 
9 using namespace std;
10 using namespace thenable;
11 
12 int main() {
13  ThenablePromise<int> p, k;
14 
15  cout << "Starting asynchronous tasks on thread: " << this_thread::get_id() << endl;
16 
17  p.then( [&k]( int i ) {
18  assert( i == 10 );
19 
20  cout << "Hello, World! from thread: " << this_thread::get_id() << endl;
21 
22  k.set_value( 421 );
23 
24  return k.get_future();
25 
26  } ).then( []( int i ) {
27  assert( i = 421 );
28 
29  cout << "Hello, again! from thread: " << this_thread::get_id() << endl;
30 
31  }, then_launch::detached );
32 
33  //Launch a new thread to resolve the original promise.
34  thread( [&p] {
35  cout << "Resolving promise on thread: " << this_thread::get_id() << endl;
36 
37  p.set_value( 10 );
38  } ).join();
39 
40  //Just to keep the program alive long enough for the promises/futures to propagate.
41  this_thread::sleep_for( 10s );
42 
43  return 0;
44 }

The output of the above will be something like:

1 Starting asynchronous tasks on thread: 1
2 Resolving promise on thread: 4
3 Hello, World! from thread: 2
4 Hello, again! from thread: 3

If THENABLE_DEFAULT_POLICY is set to std::launch::deferred, then the last two actions will be performed on the same thread.

Dependencies

This project relies on files from my function_traits project located here: function_traits.

Clone the repo or download a ZIP of it and add the 'include' directory to your build system.

For example, to add it to CMakeLists.txt you would do this:

1 include_directories(
2  SYSTEM
3  "/path/to/function_traits/include"
4  "/path/to/thenable/include"
5 )
